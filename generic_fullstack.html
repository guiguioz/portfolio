Projet Fullstack API

Projet en VueJS qui consistait à solliciter une API avec la méthode GET puis de créer avec différentes routes un site de figurine et carte amiibo.

Le but était de mettre en pratique nos connaissance en VueJS pour ensuite faire un call a l'API utiliser vue router ainsi que de créer les composants nécéssaires à l'application.
